# Panduan Berkontribusi

Terima kasih atas minat Anda untuk berkontribusi pada PawnStudio! Kami sangat menghargai setiap bantuan, baik itu berupa kode, laporan bug, maupun saran fitur.

## Cara Berkontribusi

### 1. Melaporkan Bug
Jika Anda menemukan bug atau aplikasi force close, silakan buka [Issues](https://github.com/Julakk/PawnStudio/issues) dan berikan informasi berikut:
* Versi Android yang digunakan.
* Langkah-langkah untuk mereplikasi bug tersebut.
* Screenshot atau rekaman layar jika diperlukan.

### 2. Memberikan Saran Fitur
Punya ide fitur baru? Jangan ragu untuk membuat Issue baru dengan label `enhancement` agar kita bisa mendiskusikannya lebih lanjut.

### 3. Kontribusi Kode (Pull Request)
Jika Anda ingin mengirimkan perbaikan kode:
1. Fork repositori ini.
2. Buat branch baru untuk fitur atau perbaikan Anda (`git checkout -b fitur-baru`).
3. Lakukan commit perubahan Anda.
4. Push ke branch Anda.
5. Buka **Pull Request** ke repositori utama.

Pastikan kode Anda bersih dan mudah dipahami. Terima kasih telah membantu PawnStudio menjadi lebih baik!
