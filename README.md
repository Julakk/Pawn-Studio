# PawnStudio

PawnStudio adalah aplikasi code editor berbasis web untuk perangkat Android yang dirancang agar ringan, responsif, dan mudah digunakan untuk pengembangan kode di mana saja.

## Fitur Utama
* **Monaco Editor Engine**: Pengalaman *coding* seperti di VS Code.
* **Mobile Optimized**: Antarmuka yang disesuaikan untuk layar sentuh.
* **Lightweight**: Fokus pada performa agar tidak membebani perangkat.

## Status Proyek
Saat ini PawnStudio sedang dalam tahap **pengembangan aktif (Pre-Alpha)**. Fitur masih terus dikembangkan dan disempurnakan.
